# Mental Health Tracker

A simple Mental Health Tracker that helps users record their daily mood, track progress, and get personalized tips to improve emotional well-being.

## Features
- Log daily moods
- Track weekly mood progress
- Access exercises and mini-games for better mental wellness
- User-friendly interface

## Technologies Used
- Java (Swing for UI)
- File I/O for data storage
- Object-Oriented Programming (OOP) concepts

## How to Run
1. Clone the repository
2. Open in your favorite IDE (like Eclipse or IntelliJ)
3. Run the `Main` class
